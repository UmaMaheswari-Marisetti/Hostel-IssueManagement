# Hostel Issue Management System
A MERN stack application that allows hostel students to raise issues and admins to manage them efficiently.

## Features
- Student & Admin Authentication
- Issue Raising & Tracking
- Issue Status Management (Pending / In Progress / Resolved)
- Role-Based Dashboards
- Secure JWT Authentication

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express.js
- Database: MongoDB

## Future Enhancements
- Bus Registration & Survey System
- Online Payments
- Analytics Dashboard
